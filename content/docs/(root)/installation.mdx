---
title: Installation
description: Install and configure brodin/ui for any framework.
---

<Steps>

### Set up Next.js

Run the following command to set up a Next.js project:

```bash
npx create-next-app@latest
```

### Set up shadcn/ui

Run the `init` command to create a new project or to setup an existing one:

```bash
npx shadcn@latest init
```

### Set up brodin/ui

Add the `app` component that will set up all required files:

```bash
npx shadcn@latest add https://ui.brodin.dev/r/app.json --yes --overwrite
```

Install the fonts and place them in the [/public/fonts/](./public/fonts/) folder:
- [iA Writer Quattro](
https://github.com/iaolo/iA-Fonts/raw/master/iA%20Writer%20Quattro/Variable/iAWriterQuattroV.ttf
)
- [iA Writer Mono](https://github.com/iaolo/iA-Fonts/raw/master/iA%20Writer%20Mono/Variable/iAWriterMonoV.ttf)

A base next.js app is now set up with the Brodin style!

### Add Components

You can now start adding components to your project.

```bash
npx shadcn@latest add https://ui.brodin.dev/r/button.json
```

The command above will add the `Button` component to your project. You can then import it like this:

```tsx {1,6} showLineNumbers title="app/page.tsx"
import { Button } from "@/components/ui/button"

export default function Home() {
  return (
    <div>
      <Button>Click me</Button>
    </div>
  )
}
```

</Steps>
