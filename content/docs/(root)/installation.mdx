---
title: Installation
description: Install and configure brodin/ui for any framework.
---

<Steps>

### Set up Next.js

Run the following command to set up a Next.js project:

```bash
npx create-next-app@latest
```

### Set up shadcn/ui

Run the `init` command to create a new project or to setup an existing one:

```bash
npx shadcn@latest init
```

### Set up brodin/ui

1. Add the `app` component that will set up all required files:

```bash
npx shadcn@latest add https://ui.brodin.dev/r/app.json --yes --overwrite
```

2. Install the fonts and place them in the [/public/fonts/](./public/fonts/) folder:
- [iA Writer Quattro](
https://github.com/iaolo/iA-Fonts/raw/master/iA%20Writer%20Quattro/Variable/iAWriterQuattroV.ttf
)
- [iA Writer Mono](https://github.com/iaolo/iA-Fonts/raw/master/iA%20Writer%20Mono/Variable/iAWriterMonoV.ttf)

- Set up the required environment variables in [.env file](./env):

```bash
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

> Don't forget to add it to Vercel as well with the real url.

3. Select and set the logo for your app:
- Choose one of [the available logo](/docs/brand#logo) and click on it to copy the install command
- Run the command to add the logo to your app
- Import it in the [app/icon.tsx](./app/icon.tsx) file instead of the existing logo.

A base next.js app is now set up with the Brodin style!

### Add Components

You can now start adding components to your project.

```bash
npx shadcn@latest add https://ui.brodin.dev/r/button.json
```

The command above will add the `Button` component to your project. You can then import it like this:

```tsx {1,6} showLineNumbers title="app/page.tsx"
import { Button } from "@/components/ui/button"

export default function Home() {
  return (
    <div>
      <Button>Click me</Button>
    </div>
  )
}
```

</Steps>
